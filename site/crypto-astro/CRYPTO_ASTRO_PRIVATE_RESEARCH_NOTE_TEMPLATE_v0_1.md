## Purpose

This artifact defines the safe template for a Crypto-Astro Private Research Note v0.1.

The template is designed for controlled private research delivery. It does not authorize client delivery, payment route, formula execution, A/E numeric activation, runtime/backend/API, TVL implementation, public alpha feed, trading signal, forecast, price target, or investment recommendation.

## Scope

This template applies to:

* private Crypto-Astro research notes
* operator-reviewed synthesis packets
* request-based asset-context interpretation
* source/proof-aware delivery
* future asset-context corridors that preserve the same boundary

This template does not expose formulas, weights, hidden thresholds, validation datasets, ORION reasoning, private client context, or runtime-generated alpha.

## Template usage boundary

Allowed use:

* private research context
* asset-specific context
* market context
* temporal / event-window context
* pattern synthesis
* uncertainty framing
* source/proof notes
* operator-reviewed interpretation

Not allowed use:

* buy / sell / hold
* entry / exit
* leverage
* price target
* guaranteed outcome
* urgent trade instruction
* investment recommendation
* public alpha table
* formula weights
* hidden thresholds
* validation dataset
* ORION reasoning exposure
